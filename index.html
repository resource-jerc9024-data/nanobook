iPhone Shortcut Blueprint

Open the native Shortcuts app on your iPhone, click the + icon in the top right corner, name it exactly "Talk to Nova", then stack these 4 actions:

  • Action 1: Dictate Text
    ↳ Captures your physical speech into a variable text string.
  • Action 2: Get Contents of URL
    • Paste the copied Webhook URL into this field.
    • Expand properties: Change method to POST.
    • Add Header: Key: Content-Type, Value: application/json.
    • Request Body: Choose JSON → Add new text field with Key: text and Value: select the Dictated Text variable.
  • Action 3: Get Dictionary Value
    ↳ Set Key to reply. Set Target to Contents of URL response payload.
  • Action 4: Speak Text
    ↳ Feeds the isolated dictionary response directly back out your device's speakers.
